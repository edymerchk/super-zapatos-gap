Super Zapatos
================

Sample Rails Application that use [Grape](https://github.com/intridea/grape) to provide a json API

Ruby on Rails
-------------

This application requires:

- Ruby 2.1.3
- Rails 4.1.6


## Live

Live code is deployed on http://super-zapatos-gap.herokuapp.com/
